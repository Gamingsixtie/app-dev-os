## Interaction Contracts

### Wizard Step Navigation (D-03)

| Interaction | Behavior |
|-------------|----------|
| Progress bar step click | Only completed steps are clickable (same as existing ProgressBar) |
| "Volgende stap" button | Validates current step, advances to next. Disabled when step is incomplete. |
| "Vorige stap" button | Returns to previous step, preserving entered data |
| Step 1 -> Step 2 transition | Triggers AI extraction (if notes entered) or skips (if "Niet bekend" clicked) |
| Step 2 -> Step 3 transition | Validates at least 1 module has a variant selected |
| Step 3 "Pas tabel aan" | Writes wizard results to comparison store, closes wizard to collapsed state |

### Step 1: Gespreksnotities (D-09, D-10, D-11)

| Element | Spec |
|---------|------|
| Textarea | Full-width, minimum 4 rows, placeholder: "Beschrijf het gesprek met de school..." |
| "Niet bekend" button | Secondary (ghost) button below textarea. Skips AI extraction, goes to step 2 with empty pre-fill. |
| AI extract trigger | Automatic on "Volgende stap" click when textarea has content. Shows inline loading with pulse animation. |
| Extraction result | Not shown in step 1 -- results populate step 2 variant selections with confidence badges. |

### Step 2: Variant-selectie (D-12, D-13, D-14)

| Element | Spec |
|---------|------|
| Module list | Vertical list of selected modules. Each module shows: name, current provider badge, variant selector. |
| Provider selector | Per module: radio group with "DIA", "JIJ", "Geen" options. Pre-filled from AI extraction or moduleSetups (D-07). |
| Variant selector (DIA) | When DIA selected: card grid showing DIA packages with name, price/student, included modules. "Aanbevolen" badge on smart-suggested package (D-14). |
| Variant selector (JIJ) | When JIJ selected: card grid showing JIJ tiers with name, annual fee, price/test, administration range. "Aanbevolen" badge on tier matching school size. |
| Confidence indicator | Per module: colored dot + label. Green = "Uit notities afgeleid", Orange = "Onzeker", Gray = "Niet bekend" (D-11). |
| "Aanbevolen" badge | Small pill badge, `bg-cito-accent/10 text-cito-accent border-cito-accent/30`, text: "Aanbevolen" |

### Step 3: AI Matching + Resultaat (D-15, D-16, D-19, D-21)

| Element | Spec |
|---------|------|
| Extra context field (D-21) | 3 labeled input fields stacked vertically: "Eventuele korting" (text), "DMU-focus" (text), "Bijzonderheden" (textarea, 2 rows). All optional. |
| "Genereer advies" button | Primary CTA (`bg-cito-accent`). Triggers AI streaming. Disabled during streaming. |
| Streaming display | Markdown-rendered text appearing character-by-character in a `bg-cito-primary/5 border-cito-primary/20` container. Blinking cursor indicator during streaming. |
| Advice result | Structured display: samenvatting banner + typed advice cards (reuse TYPE_CONFIG pattern from AdvicePanel). Cito bundle recommendation highlighted. |
| Editable result | Each module's matching can be toggled/adjusted inline before confirming (D-19). Mutable copy pattern from DiffView.tsx. |
| "Pas tabel aan" button | Primary CTA (`bg-cito-accent`), full width, text: "Pas tabel aan". Only enabled after advice is generated. Explicit confirmation per D-24. |
| AI disclaimer | 11px gray text below results: "Dit advies is automatisch gegenereerd. Controleer de informatie altijd voor gebruik in een gesprek." |

### Scenario Detection (D-26)

| Scenario | Banner Style | Copy |
|----------|-------------|------|
| Deels concurrent (default) | No banner, wizard proceeds normally | — |
| Alles oud-Cito | `bg-blue-50 border-blue-200` info banner | "Deze school gebruikt uitsluitend het huidige Cito-platform. Ga naar Migratie voor een overstap-businesscase." + link |
| Alles nieuw-Cito | `bg-emerald-50 border-emerald-200` info banner | "Deze school gebruikt al het nieuwe Cito-platform. Wat wilt u vergelijken?" + option to proceed anyway |

### Wizard Collapsed State (D-04)

| State | Display |
|-------|---------|
| Not started | Wizard is fully visible, step 1 active |
| Completed (after "Pas tabel aan") | Wizard collapses to a summary bar: "AI vergelijkingsadvies actief" + "Opnieuw doorlopen" button |
| Re-running | Wizard expands back to step 1, preserving previous inputs as defaults |

## Loading & Streaming States

| State | Visual |
|-------|--------|
| Step 1 AI extraction in progress | "Volgende stap" button shows spinner + "Analyseren..." text. Textarea disabled. Pulse animation on button. |
| Step 3 AI streaming in progress | Streaming container visible with text appearing incrementally. Blinking cursor (CSS `animate-pulse` on a `|` character). "Genereer advies" button disabled, shows spinner + "Genereren..." |
| Step 3 streaming complete | Cursor disappears. Advice cards render. "Pas tabel aan" button becomes enabled. |
| Network error mid-stream | Streaming stops. Error message appears inline below partial content. "Opnieuw genereren" button appears. |
