# Phase 10.2 Plan 01: Provider-Specific Price Calculators Summary

**Provider-aware calculators (Cito bundles, DIA volume discounts, JIJ tiered-license) with TDD, refactored calculateComparison producing per-module breakdowns, parity-verified against old pipeline for 4 school sizes**

## What Was Built

### Task 1: Calculator Types, Implementations, and Factory (TDD)

Created 6 new files implementing the `ProviderPriceCalculator` interface:

- **FlatCalculator** (SAQI): Simple per-student pricing with Dutch breakdown labels
- **JijCalculator**: Tiered-license pricing varying with school size -- selects correct tier based on totalStudents * defaultTestsPerStudent, computes licenseCostPerStudent + testCostPerStudent
- **DiaCalculator**: Package-bundle pricing with volume discounts (0%/5%/10%) and package optimization via existing selectOptimalDiaPackage
- **CitoCalculator**: Platform+module pricing with bundle logic (individual/basis/plus), checks bundle eligibility before applying bundle pricing
- **createCalculator factory**: Exhaustive switch on pricingStrategy.type, passes citoBundleType and selectedModules to CitoCalculator

25 TDD tests covering all calculators, override precedence, Dutch labels, tier selection.

### Task 2: Refactored calculateComparison() with Parity Tests (TDD)

Refactored `calculateComparison()` to use provider calculators internally:

- **Overloaded signature**: accepts either `PriceRecord[]` (legacy, routes to calculateComparisonLegacy) or `ComparisonOptions` (new, uses calculators)
- **Extended ProviderCost**: added `breakdown`, `isPackagePrice`, `packageId`, `tierId` fields (additive, backward-compatible)
- **Extended ComparisonResult**: added `diaPackageResult: DiaPackageResult | null` field
- **Backward compatibility**: all existing callers (store, schijnvoordeel, DashboardTab, etc.) continue working via the PriceRecord[] overload

10 parity tests verify numerical equivalence between old pipeline (applyDynamicJijPrices -> applyDiaVolumeDiscount -> calculateComparisonLegacy) and new calculateComparison for school sizes 100, 500, 800, 1200.
