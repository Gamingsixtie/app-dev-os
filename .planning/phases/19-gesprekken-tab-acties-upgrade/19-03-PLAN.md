**2. Inline Title Edit (D-12, IC-04):**
- Add state: `const [isEditingTitle, setIsEditingTitle] = useState(false);` and `const [editTitle, setEditTitle] = useState(action.title);`
- Add ref: `const titleInputRef = useRef<HTMLInputElement>(null);`
- Get mutation: `const updateActionMutation = useUpdateAction();`
- When `isEditingTitle` is true, render an `<input>` instead of the `<p>` title:
  ```tsx
  {isEditingTitle ? (
    <input
      ref={titleInputRef}
      type="text"
      value={editTitle}
      onChange={e => setEditTitle(e.target.value)}
      onBlur={() => {
        if (editTitle.trim() && editTitle !== action.title) {
          updateActionMutation.mutate({ schoolId, actionId: action.id, data: { title: editTitle.trim() } });
        } else {
          setEditTitle(action.title);
        }
        setIsEditingTitle(false);
      }}
      onKeyDown={e => {
        if (e.key === 'Enter') (e.target as HTMLInputElement).blur();
        if (e.key === 'Escape') { setEditTitle(action.title); setIsEditingTitle(false); }
      }}
      className="w-full text-sm font-semibold text-neutral-700 border border-cito-primary rounded px-1 focus:outline-none focus:ring-1 focus:ring-cito-primary"
      placeholder="Actietitel..."
    />
  ) : (
    <p
      onClick={() => { setIsEditingTitle(true); setEditTitle(action.title); }}
      className="text-sm font-semibold text-neutral-700 cursor-text"
    >
      {action.title}
    </p>
  )}
  ```
- Auto-focus + select text on mount: `useEffect(() => { if (isEditingTitle && titleInputRef.current) { titleInputRef.current.focus(); titleInputRef.current.select(); } }, [isEditingTitle]);`
