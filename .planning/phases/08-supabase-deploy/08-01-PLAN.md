Create `supabase/migrations/002_rls_policies.sql`:

1. Enable RLS on ALL 8 tables:
```sql
ALTER TABLE teams ENABLE ROW LEVEL SECURITY;
ALTER TABLE users ENABLE ROW LEVEL SECURITY;
ALTER TABLE schools ENABLE ROW LEVEL SECURITY;
-- ... all 8 tables
```

2. Security definer helper functions (per Pitfall 1 from RESEARCH.md):
```sql
CREATE OR REPLACE FUNCTION get_user_team_id()
RETURNS UUID LANGUAGE sql SECURITY DEFINER SET search_path = public AS $$
  SELECT team_id FROM users WHERE id = auth.uid();
$$;

CREATE OR REPLACE FUNCTION get_user_role()
RETURNS TEXT LANGUAGE sql SECURITY DEFINER SET search_path = public AS $$
  SELECT role FROM users WHERE id = auth.uid();
$$;
```

3. Policies per table:
- `users`: SELECT where team_id = get_user_team_id() OR id = auth.uid(). UPDATE only own record (id = auth.uid()).
- `teams`: SELECT where id = get_user_team_id().
- `schools`: SELECT where team_id = get_user_team_id(). INSERT where owner_id = auth.uid() AND get_user_role() = 'accountmanager'. UPDATE where owner_id = auth.uid() AND get_user_role() = 'accountmanager'. DELETE where owner_id = auth.uid() AND get_user_role() = 'accountmanager'.
- `contacts`, `conversations`, `actions`, `system_events`, `school_prices`: SELECT where school_id IN (SELECT id FROM schools WHERE team_id = get_user_team_id()). INSERT/UPDATE/DELETE where school_id IN (SELECT id FROM schools WHERE owner_id = auth.uid()) AND get_user_role() = 'accountmanager'.
