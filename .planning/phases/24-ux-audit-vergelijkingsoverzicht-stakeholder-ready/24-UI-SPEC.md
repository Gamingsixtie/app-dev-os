# Phase 24 — UI Design Contract

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps (`gap-1`), inline padding (`px-1`, `py-0.5`) |
| sm | 8px | Compact element spacing (`gap-2`, `p-2`), toolbar internal gaps |
| md | 16px | Default element spacing (`gap-4`, `p-4`, `px-4`), inner content padding on mobile |
| lg | 24px | Section heading margin-bottom (`mb-6`), card internal padding |
| xl | 32px | Section vertical padding (`py-8`), layout gaps between major blocks |
| 2xl | 48px | Page-level top/bottom padding (`py-12`) |

Exceptions: `max-w-[960px]` content container width is the established page constraint. Each color band section manages its own inner container at this width.

## Typography

| Role | Size | Weight | Line Height | Tailwind Class |
|------|------|--------|-------------|----------------|
| Body | 14px | 400 (regular) | 1.5 | `text-sm` |
| Label | 11px | 400 (regular) | 1.4 | `text-[11px]` |
| Section heading | 16px | 600 (semibold) | 1.3 | `text-base font-semibold` |
| Page title | 20px | 600 (semibold) | 1.2 | `text-xl font-semibold` |

Weights used: **2** -- 400 (regular) for body text and labels, 600 (semibold) for section headings and page title. Weight contrast plus size difference provides sufficient visual hierarchy without additional weight steps.

Source: Adapted from existing PriceComparisonPage component patterns. Body uses `text-sm` (14px). Section headings bumped from 15px to 16px (`text-base`) for clear size differentiation from body text (2px delta instead of 1px). Labels remain 11px for auxiliary metadata. Page title at 20px provides top-level hierarchy via size alone.

## Section Order Contract (D-05)

The page renders sections in this exact order. Each section is wrapped in a `SectionBand` component with alternating background color.

```
1. AI Advies Hero        — collapsed by default (D-09), 2-3 line summary visible
2. Bundel/Periode        — CitoBundleSelector + DiaBundleSelector + PeriodToggle (D-02)
3. Totaal-kaarten        — ComparisonSummary WITHOUT differentiators (D-03, D-06)
4. Provider Toolbar      — merged ProviderSelector + PricingModelCards (D-08)
5. ComparisonTable       — existing detail panels preserved (D-12)
6. ComparisonChart       — collapsed by default (D-10)
7. MeerwaardePanel       — TimeSavings + BusinessCaseCTA only, collapsed by default (D-07, D-11)
8. DisclaimerFooter      — unchanged
```

## Progressive Disclosure Contract

| Section | Default State | Trigger | Mechanism |
|---------|---------------|---------|-----------|
| AI Advies Hero | Collapsed: 2-3 line summary visible | "Lees volledig advies" button | Controlled React state (`useState`) -- needs custom summary view |
| ComparisonChart | Collapsed: only heading + chevron | Click on `<summary>` | Native `<details>/<summary>` |
| MeerwaardePanel | Collapsed: only heading + chevron | Click on `<summary>` | Native `<details>/<summary>` |
| ComparisonTable rows | Collapsed per row | Click on row | Existing expandable detail-panel pattern (D-12, unchanged) |

### Collapse/Expand Interaction

- Chevron icon rotates 180 degrees on open: `transition-transform duration-200 group-open:rotate-180`
- No spring physics, no framer-motion. CSS transitions only.
- `<details>` elements use Tailwind `group` class for `group-open:` styling
- AI advies expand button uses `bg-cito-accent text-white` styling, consistent with existing CTA pattern

## Component Contracts

### SectionBand (new utility component)

```
Props: bg ('bg-white' | 'bg-neutral-50'), children
Renders: <section className="{bg} py-8"><div className="max-w-[960px] mx-auto px-4 sm:px-8">{children}</div></section>
```

### ProviderToolbar (new merged component, D-08)

```
Layout: horizontal flex row with flex-wrap
Content: "Vergelijk:" label + provider checkboxes with color dots + info popover buttons
Mobile: wraps to 2 rows on screens < 640px
Info trigger: click opens popover (not hover -- must be touch-friendly per D-14)
Popover: 256px wide, absolute positioned, white bg, neutral-200 border, rounded-lg, shadow-lg, 12px body text
Popover dismiss: click outside closes
```

### AI Advies Hero (restructured, D-01 + D-09)

```
Contains: SchoolplanBanner content (integrated, no separate banner) + wizard CTA + analysis summary
Collapsed state: 2-3 line conclusion text + "Lees volledig advies" expand button
Expanded state: full analysis content (existing AnalysisPanel)
No analysis yet state: wizard CTA only (no empty summary text)
```

### ComparisonSummary (modified, D-06)

```
Removed: DifferentiatorsSection / "Unieke Cito voordelen" list
Kept: totalen per aanbieder grid with verschilbedragen
```

### MeerwaardePanel (modified, D-07)

```
Removed: DifferentiatorsSection
Kept: TimeSavingsSection + BusinessCaseCTA
Wrapped in: <details> (collapsed by default)
```

## Accessibility Requirements

- All `<details>/<summary>` are keyboard navigable by default (native HTML)
- Info popover buttons have `aria-label="Prijsmodel {provider}"`
- Provider checkboxes have associated `<label>` elements
- Color is never the sole differentiator -- provider names always accompany color dots
- Minimum touch target: 44x44px for all interactive elements on mobile
- Popover dismiss on Escape key
