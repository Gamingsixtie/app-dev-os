## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding, `gap-1` |
| sm | 8px | Compact element spacing, kanban card inner gap, `gap-2` / `p-2` |
| md | 16px | Default element spacing, form field gaps, `gap-4` / `p-4` |
| lg | 24px | Section padding, form container padding, `p-6` |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Not used in this phase |
| 3xl | 64px | Not used in this phase |

Exceptions: Touch targets use 44px minimum height (`h-[44px]`) per existing project convention (drag handles, buttons, inputs).

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px (`text-base`) | 400 (normal) | 1.5 |
| Label | 14px (`text-sm`) | 600 (semibold) | 1.4 |
| Heading | 20px (`text-[20px]`) | 600 (semibold) | 1.2 |
| Small / Meta | 12px (`text-xs`) | 400 (normal) | 1.4 |

**Scale rationale:** 4 tiers with clear separation — 12 / 14 / 16 / 20. The previous 13px tier was collapsed to 12px to create a visible 2px step from the 14px Label tier. 12px is used for timestamps, deadline dates, action type chips, and overdue badges.

**Source:** Existing `ConversationForm.tsx` and `ActionItem.tsx` patterns, adjusted from 13px to 12px for scale clarity.

## Interaction Contracts

### IC-01: Microphone Toggle (D-01, D-02, D-03, D-04)

| State | Visual | Behavior |
|-------|--------|----------|
| Idle (supported) | Microphone SVG icon, `text-neutral-500`, `hover:text-neutral-700` | Click starts recording |
| Recording | Microphone SVG icon, `text-cito-accent`, pulsing `ring-2 ring-cito-accent/50` animation | Click stops recording. Transcribed text appends to textarea value. |
| Unsupported | Microphone SVG icon, `text-neutral-300`, `cursor-not-allowed`, `opacity-50` | Disabled. Hover shows tooltip per D-04. |

**Position:** Right side of the "Inhoud" label row, aligned with label text. Same vertical line as the label.
**Animation:** `animate-pulse` on the ring only (not the icon itself). CSS: `@keyframes pulse-ring`.

### IC-02: Contact Dropdown (D-06, D-07)

| Aspect | Specification |
|--------|--------------|
| Element | Native `<select>` (existing pattern) |
| Option format | `{name} -- {DMU_POSITION_LABELS[dmuPosition]} -- {ENGAGEMENT_STATUS_LABELS[engagementStatus]}` |
| Required | Yes. Zod schema enforces non-empty `contactId` |
| Empty state | Single disabled option: "Geen contactpersonen" |
| Styling | `h-[44px]` height, `border-neutral-200`, `focus:ring-2 focus:ring-cito-primary` (existing) |

### IC-03: Always-Visible Inline Input (D-09)

| Aspect | Specification |
|--------|--------------|
| Position | Bottom of "Te doen" column, always visible (no toggle) |
| Element | `<input type="text">` |
| Placeholder | "Nieuwe actie..." |
| Submit | Enter key creates action with `status: 'todo'` |
| Clear | Input resets to empty after successful creation |
| Escape | Clears input text (does not hide the input) |
| Height | `h-[36px]` (compact, below cards) |
| Border | `border border-neutral-200 rounded-lg` |
| Focus | `focus:ring-2 focus:ring-cito-primary` |

### IC-04: Inline Title Edit (D-12)

| Aspect | Specification |
|--------|--------------|
| Trigger | Click on action title text |
| Element | `<input type="text">` replaces `<p>` |
| Auto-save | On blur OR Enter key |
| Cancel | Escape key reverts to original value |
| Visual | Same font size (14px semibold) as display mode. Border appears only in edit mode: `border border-cito-primary rounded px-1` |
| Focus | Auto-focus on mount with text selected |

### IC-05: Action Type Label (D-10)

| Aspect | Specification |
|--------|--------------|
| Display | Colored chip/badge below the action title |
| Size | `text-xs px-2 py-1 rounded-full` |
| Set via | Click "Type" link on card (appears on hover) -> dropdown with preset options + custom input |
| Preset options | bellen, mailen, offerte, intern overleg |
| Custom | Free text typed into the same dropdown input |
| Not during creation | Type is set after the action exists, not in the inline input (D-10) |

### IC-06: Deadline Field (D-13)

| Aspect | Specification |
|--------|--------------|
| Display | Below title + type, icon + date text |
| Element | Native `<input type="date">` (consistent with ConversationForm date field) |
| Trigger | Click calendar icon or "Deadline" placeholder text on the card |
| Overdue | Card gets `border-red-300`. Date text becomes `text-red-600 font-semibold`. |
| Clear | Small X button next to date to remove deadline. `aria-label="Deadline verwijderen"` on the button element. |
| Format | `nl-NL` locale date formatting via `Intl.DateTimeFormat` |

### IC-07: Delete Confirmation Modal (D-14)

| Aspect | Specification |
|--------|--------------|
| Overlay | `fixed inset-0 z-50`, `bg-black/50` backdrop |
| Card | `bg-white rounded-lg p-8 max-w-[420px] w-full mx-4 shadow-xl` |
| Role | `alertdialog` with `aria-describedby` |
| Title | `text-[20px] font-semibold text-neutral-900 mb-4` |
| Body | `text-base text-neutral-600 mb-6` |
| Cancel button (conversation) | Label: "Gesprek bewaren". Style: `h-[44px] px-4 text-sm font-semibold text-neutral-700 hover:bg-neutral-100 rounded-lg` |
| Cancel button (action) | Label: "Actie bewaren". Style: `h-[44px] px-4 text-sm font-semibold text-neutral-700 hover:bg-neutral-100 rounded-lg` |
| Confirm button | `h-[44px] px-4 text-sm font-semibold bg-red-600 text-white rounded-lg hover:bg-red-700` |
| Button layout | `flex justify-end gap-4` |
| Dismiss | Click backdrop or cancel button |

**ConfirmDialog props:** The `ConfirmDialog` component accepts a `cancelLabel` prop (string) so the caller specifies the context-specific cancel copy. No default "Annuleren" fallback.

**Source:** Exact layout of `DeleteSchoolDialog.tsx` pattern, with context-specific cancel labels.

### IC-08: Hidden AI Intake (D-05)

| Aspect | Specification |
|--------|--------------|
| IntakeModeToggle render | Removed from JSX (import kept) |
| intakeMode state | Hard-coded to `'manual'`, no setter |
| AI code paths | Remain compilable but never execute |
| Visual result | Form opens directly in manual mode with no toggle visible |

### IC-09: Kanban Visual Refinements (D-16)

| Aspect | Specification |
|--------|--------------|
| Column headers | Add inline SVG icons before label text. Te doen: clipboard icon. In uitvoering: play-circle icon. Afgerond: check-circle icon. |
| Card hover | `hover:shadow-sm hover:border-neutral-300 transition-all` |
| Card spacing | `p-3` inner padding (existing), `gap-2` between cards in column |
| Conversation link indicator | Existing clock icon + date. No change needed. |
| Created date | Add `text-xs text-neutral-400` date below title (new) |
