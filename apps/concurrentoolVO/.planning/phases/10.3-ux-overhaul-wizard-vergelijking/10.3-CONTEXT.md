<domain>
## Phase Boundary

Overzichtelijke, flexibele wizard en vergelijkingsweergave met dynamische provider-kolommen, prijsmodel-uitleg en prijsopbouw. Bouwt voort op de uitgebreide MODULE_CATALOG (Phase 10.1) en provider-aware engine met breakdowns (Phase 10.2).

<decisions>
## Implementation Decisions

### Module-selectie Wizard
- Category-grouped cards (LVS / Overige instrumenten) consistent with current WizardStep3 pattern, now showing 10 modules
- Provider-beschikbaarheid badges: small colored dots per provider (Cito=blauw #003082, DIA=oranje, JIJ=groen) on each module card using availableFrom data
- Quick-pick combinaties: pre-defined combos "LVS Basis" (RE+NL+EN), "LVS Compleet" (alle LVS modules), "Alles" — one-click select
- Frans/Duits/Spaans grouped as subcategory "Moderne Vreemde Talen" within Overige instrumenten — JIJ-only badge visible

### Vergelijkingstabel & Provider-kolommen
- Dynamic provider selection: checkbox row above table, user picks which providers to compare (default: Cito + providers from school productgebruik)
- Column ordering: Cito always first (our product), then alphabetical — consistent with existing ComparisonTable
- Prijsmodel-uitleg: collapsible card per provider above the table explaining their pricing model ("DIA: pakketprijs-model — 6 pakketten, prijs per pakket")
- Schoolgrootte-impact: badge in table header showing active tier/pakket ("JIJ: Licentie 3" / "DIA: Pakket compleet") — updates dynamically with school size

### Prijsopbouw & Detail-panel
- Price breakdown: expandable row per module showing step-by-step calculation from engine breakdown data (Dutch labels from Phase 10.2)
- Breakdown depth: full calculation chain ("Basisprijs → Bundel (Basis) → €7.82/ll × 500 = €3.910") with each step labeled
- Cito bundle selector: inline toggle in Cito column header (Individual/Basis/Plus) — recomputes live, shows which modules are bundled

### Claude's Discretion
- Exact responsive behavior on tablet for the expanded table
- Animation/transition timing for expand/collapse
- Provider color palette for DIA and JIJ (Cito = #003082)
- Quick-pick button styling and placement
- Checkbox row visual design for provider selection
- Loading state during recomputation after bundle/provider changes

</decisions>
