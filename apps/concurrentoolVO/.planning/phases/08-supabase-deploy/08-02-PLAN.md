Create `src/features/auth/AuthProvider.tsx`:

1. Define `AuthContextType` interface:
```typescript
interface AuthContextType {
  user: User | null;          // from @supabase/supabase-js
  session: Session | null;    // from @supabase/supabase-js
  userProfile: UserProfile | null;  // from custom users table
  loading: boolean;
  signIn: (email: string, password: string) => Promise<{ error: string | null }>;
  signInWithMagicLink: (email: string) => Promise<{ error: string | null }>;
  signOut: () => Promise<void>;
}
```

2. Create React context with `createContext<AuthContextType>`.

3. `AuthProvider` component:
   - State: user, session, userProfile, loading (initial true)
   - On mount: call `supabase.auth.getSession()` to restore existing session
   - Subscribe to `supabase.auth.onAuthStateChange((event, session) => {...})` — update user/session state. On SIGNED_IN or TOKEN_REFRESHED: fetch user profile from `users` table via `supabase.from('users').select('*').eq('id', session.user.id).single()`. Map snake_case to camelCase for UserProfile (team_id -> teamId).
   - On SIGNED_OUT: clear user, session, userProfile
   - Set loading=false after initial session check
   - Cleanup: unsubscribe from onAuthStateChange on unmount

4. `signIn` method: calls `supabase.auth.signInWithPassword({ email, password })`. Returns `{ error: null }` on success, `{ error: "Onjuist e-mailadres of wachtwoord. Probeer het opnieuw." }` on AuthApiError, `{ error: "Er is iets misgegaan. Probeer het later opnieuw." }` on other errors, `{ error: "Te veel inlogpogingen. Probeer het over een paar minuten opnieuw." }` on rate limit (status 429).

5. `signInWithMagicLink` method: calls `supabase.auth.signInWithOtp({ email })`. Returns `{ error: null }` on success. Same error messages pattern.

6. `signOut` method: calls `supabase.auth.signOut()`. Clears state.

7. Export `useAuth()` custom hook that calls `useContext(AuthContext)` with a guard throwing if used outside provider.

Also re-export `useAuth` from `hooks.ts` for convenient imports.

Create `src/features/auth/ProtectedRoute.tsx`:
```typescript
import { useAuth } from './AuthProvider';
import { AuthLoadingScreen } from './AuthLoadingScreen';

export function ProtectedRoute({ children }: { children: React.ReactNode }) {
  const { user, loading } = useAuth();
  if (loading) return <AuthLoadingScreen />;
  if (!user) {
    // Redirect to login — use window.location or navigate from TanStack Router
    window.location.href = '/login';
    return null;
  }
  return <>{children}</>;
}
```
