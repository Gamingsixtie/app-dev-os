```typescript
// Inline type for Dexie data -- avoids conflict with parallel Plan 08-03
// which changes SchoolRecord.id from number to string
interface DexieSchoolRecord {
  id?: number;
  slug: string;
  name: string;
  createdAt: Date;
  updatedAt: Date;
  isComplete: boolean;
  completedSteps: number[];
  levels: string[];
  studentCounts: Record<string, Record<number, number>>;
  selectedModules: string[];
  moduleSetups: Array<{ moduleId: string; currentProvider: string; currentPrice?: number; source?: string }>;
  scenario: string | null;
  appliedOverrides: Array<{ moduleId: string; provider: string; price: number; source: string }>;
  migrationHourlyRate: number;
  migrationTimeSavingOverrides: Record<string, number>;
  contacts: Array<Record<string, unknown>>;
  conversations: Array<Record<string, unknown>>;
  actions: Array<Record<string, unknown>>;
  systemEvents: Array<Record<string, unknown>>;

```

Use `DexieSchoolRecord` when typing `db.schools.toArray()` results. This ensures the migration code compiles regardless of whether Plan 08-03 has already changed `SchoolRecord`.

`migrateIndexedDBToSupabase(onProgress: (p: MigrationProgress) => void): Promise<MigrationResult>`:

1. Check if already migrated: `if (localStorage.getItem('supabase-migration-complete') === 'true') return { success: true, total: 0, migrated: 0, errors: [] };`
2. Read all schools from Dexie: `const localSchools = await db.schools.toArray() as DexieSchoolRecord[];`
3. If empty: mark complete, return success with total=0
4. Get current user via `supabase.auth.getUser()`, get team_id from users table
5. For each school (loop with try/catch per school):
   a. Call `onProgress({ status: 'migrating', current: i+1, total: localSchools.length, currentSchoolName: school.name })`
   b. Insert school into schools table: map all wizard fields (levels, studentCounts->student_counts, selectedModules->selected_modules, moduleSetups->module_setups, scenario, migrationHourlyRate->migration_hourly_rate, migrationTimeSavingOverrides->migration_time_saving_overrides, pipelineStatus->pipeline_status, lostDealInfo->lost_deal_info, region, tags, viewPreference->view_preference). Set owner_id, team_id, created_by, updated_by to current user.
   c. If school.contacts.length > 0: batch insert into contacts table. Map: name, dmuPosition->dmu_position, jobTitle->job_title, email, phone, preferredChannel->preferred_channel, authority, lastContactDate->last_contact_date, notes, isPrimary->is_primary, createdAt->created_at. Set school_id, created_by.
   d. If school.conversations.length > 0: batch insert into conversations table. Map: date, contactId needs to be resolved (map old string UUID to new contact UUID from step c -- store a contactIdMap from old->new). content, tags, createdAt->created_at, updatedAt->updated_at. Set school_id, created_by, updated_by. NOTE: since contacts were inserted with gen_random_uuid(), we need to insert contacts first, read back IDs, build the map, then insert conversations with mapped contact_ids. Strategy: insert contacts with `.select()` to get back the new rows with IDs. Map by contact name+school combo.
   e. If school.actions.length > 0: batch insert into actions table. Same contact_id mapping needed for conversation_id linkage. Map conversationId from old to new.
   f. If school.systemEvents.length > 0: batch insert into system_events table.
   g. Migrate appliedOverrides to school_prices (per D-05): for each override, insert into school_prices with price_type='agreed', is_active=true, source='Gemigreerd uit v1', activation_reason='Automatische migratie', activated_at=now().
   h. On error: push { schoolName: school.name, error: String(err) } to errors array
6. If all succeeded: `localStorage.setItem('supabase-migration-complete', 'true')`
7. Return MigrationResult with counts

`hasLocalData(): Promise<boolean>`:
- Try to read db.schools.count(). Return count > 0.
- Wrap in try/catch (IndexedDB might not exist). Return false on error.

`isMigrationComplete(): boolean`:
- Return `localStorage.getItem('supabase-migration-complete') === 'true'`
