| T-27-16-04 | Tampering | promptOverride malicious instructions | mitigate | Template-substitution server-side (replace template var, niet eval) + manager-only UI gate + Zod schema parse op AI-response — worst case unbruikbare response. |
| T-27-16-05 | Information Disclosure | Cross-team team_settings leak via API call | mitigate | Vercel function leest team_id uit JWT → query team_settings WHERE team_id = jwt_team_id. RLS dubbel als defense in depth. |
