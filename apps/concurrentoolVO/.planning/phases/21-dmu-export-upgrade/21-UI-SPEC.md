## Component Inventory

### New Web Components

| Component | Location | Purpose |
|-----------|----------|---------|
| AssumptionsEditor | `src/features/export/components/AssumptionsEditor.tsx` | Inline section in ExportConfigPanel for editing DMU assumptions text and focus areas |

**AssumptionsEditor interaction contract:**
- Renders below the DMU target radio group in ExportConfigPanel
- Shows editable textarea for intro text (4 rows, full width)
- Shows editable list of focus areas (each as a text input)
- "Standaard herstellen" text button resets to defaults from `dmu-assumptions.ts`
- Resets automatically when DMU target changes (D-02)
- Uses `useState` in ExportTab, not Zustand (D-02, session-scoped)
- Visual style: same card pattern as radio options (`p-3 rounded-lg border border-neutral-200`)
- Textarea: `text-sm`, neutral-700 text, neutral-200 border, 100% width
- Focus area inputs: `text-xs`, inline list with 8px vertical gap

### New PDF Components

| Component | Location | Purpose |
|-----------|----------|---------|
| CoverPage | `src/features/export/pdf/components/CoverPage.tsx` | Full A4 page with Cito logo, school name, date, DMU target, report type |
| IntroSection | `src/features/export/pdf/components/IntroSection.tsx` | Role-specific intro paragraph based on editable assumptions |
| ProductInfoSection | `src/features/export/pdf/components/ProductInfoSection.tsx` | Tagged Cito product advantages with source citations |

**CoverPage layout contract:**
- Full `<Page size="A4">` element, rendered before content pages in ReportDocument
- Vertical flex layout, centered content
- Top: Cito logo via `<Image>` component (max width 180px, max height 60px, centered)
- If logo file missing: skip Image, render "Cito" as text in 36pt Helvetica-Bold, primary color
- Middle: report title (18pt bold primary), school name (28pt bold, dark text), single accent-colored horizontal rule (3pt, #FF6600, 60px wide, centered)
- Bottom third: date in 12pt muted, DMU target in 12pt muted, confidential notice in 9pt italic accent
- Page padding: 40px (matches content pages)

**IntroSection layout contract:**
- Rendered immediately after SummarySection (before data sections)
- Single paragraph, 11pt Helvetica, line-height 1.6, neutral text-dark color
- Left margin 0 (flush with section content)
- Bottom margin 16px (before next section)
- No section title -- flows as narrative introduction

**ProductInfoSection layout contract:**
- Rendered after price/value sections, before schoolplan section
- Section title: "Relevante Cito-voordelen" in existing sectionTitle style (14pt bold primary)
- Each advantage: bold advantage text (10pt bold) + context paragraph (10pt normal) + source citation (8pt italic muted)
- Advantages separated by 8px vertical gap
- Only shows advantages matching the current DMU target's tags (via `filterByDmuTags`)
- If no advantages match after filtering: section not rendered (return null)

### Extended Components

| Component | Change |
|-----------|--------|
| ExportConfigPanel | Add AssumptionsEditor section below DMU target selection |
| ExportPreview | Add intro paragraph preview and product info preview sections |
| ReportDocument | Insert CoverPage before content, IntroSection after summary, ProductInfoSection before schoolplan |
| SchoolplanSection | Filter opportunities by DMU tags; render nothing if all filtered out |

**ExportPreview visual hierarchy note:** The preview panel's existing top-to-bottom layout (summary at top, sections below) naturally absorbs the two new sections. The intro paragraph slots in directly after the summary as a text block with no heading, maintaining the summary's visual primacy. The product info preview appears further down as a distinct section with its own "Relevante Cito-voordelen" heading. No new focal point or visual weight shift is introduced -- the summary remains the primary element the user scans first, with new sections following the established descending-importance reading order.

## Interaction States

### AssumptionsEditor

| State | Visual |
|-------|--------|
| Default | Textarea with default intro text, focus areas list with defaults |
| Editing | Standard textarea focus ring (ring-2 ring-cito-primary/20) |
| Reset available | "Standaard herstellen" button visible when content differs from defaults |
| Reset unavailable | "Standaard herstellen" button hidden when content matches defaults |
| DMU target changed | Content resets to new target's defaults (automatic, no animation) |

### ExportConfigPanel (existing, extended)

| State | Visual |
|-------|--------|
| DMU target selected | Blue border + light blue background on selected card (existing pattern) |
| Assumptions section | Always visible below DMU selection when any DMU target is selected |

### CoverPage (PDF)

| State | Visual |
|-------|--------|
| Logo available | Cito logo image rendered centered at top |
| Logo missing | "Cito" text fallback in 36pt bold primary |

### ProductInfoSection (PDF)

| State | Visual |
|-------|--------|
| Advantages exist for DMU | Section renders with filtered advantages |
| No advantages for DMU | Section omitted entirely |

### SchoolplanSection (PDF, extended)

| State | Visual |
|-------|--------|
| Opportunities exist after DMU filter | Filtered list renders |
| No opportunities after DMU filter | Section omitted entirely |
| No schoolplan uploaded | Summary includes: "Upload een schoolplan voor een nog specifiekere onderbouwing." |
