  - SchoolplanSummary with summary={analysis.summary}

- SchoolplanUpload metadata bar
- Error card: `bg-red-50 border border-red-200 rounded-lg p-6 mt-4 text-center`

**Handlers:**

`handleUpload(file: File)`:
1. setIsUploading(true), setUploadError(null), setStreamingStep(1)
2. Call `uploadAndAnalyzeSchoolplan(schoolId, file, (step, label) => setStreamingStep(step))`
3. On success: setStreamingStep(3), then after 1s delay invalidate the query (React Query refetch) and reset streamingStep to 0
4. On error: setUploadError(error.message), setStreamingStep(0)
5. Finally: setIsUploading(false)

`handleReplace()`:
1. setShowReplaceConfirm(true)

`handleConfirmReplace()`:
1. Call deleteSchoolplanAnalysis mutation
2. setShowReplaceConfirm(false)
3. Reset all state to empty

`handleAnnotationChange(index: number, annotation: OpportunityAnnotation)`:
1. Call updateAnnotation mutation with { opportunityIndex: index, annotation }
